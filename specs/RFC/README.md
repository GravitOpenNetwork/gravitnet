# Gravit RFCs

This directory contains the specifications for the Verifiable Convergence Protocol (VCP) and security models of the Gravit Open Network.

## Active Specs

- **[RFC-0001: Verifiable Convergence Protocol (VCP) v0.1](file:///root/gravit/gravitnet/specs/RFC/0001-vcp.md)**: Core interface and data structures for epistemic coordination.
- **[GQRVP Security Model](file:///root/gravit/gravitnet/specs/RFC/GQRVP-security.md)**: Formal mathematical definitions and Byzantine resilience proofs.

All RFCs are in draft and follow the standards track.
